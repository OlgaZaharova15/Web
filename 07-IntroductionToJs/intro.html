Вывод на экран

  1. Модальные окна:
    • alert(); //отображение модального окна сообщения (прям как MessageBox() :-)
    • prompt(); //модальное окно со сторокой ввода. Функция возвращает введенную строку
    • confirm();//модальное окно "Да", "Нет" (наверное)
  2. Вывод в консоль браузера: 
    console.log(.........)
  3. document.write();//выводит данные непосредственно на страницу;DEPRECATED
  4. Вывод на страницу при помощи DOM - Document Object Model, методами InnerHTML(), OuterHTML()....

Строки

  1. "string";
  2. 'string';
  3. `Interpolated string`;

Переменные и типы данных

  1. var - объявляет переменую. Создает глобольную переменную, независимо от области вилимости, в которой она объявлена;
  2. let - объявляет переменую. Переменные созданные при помощи let соществуют только в той области видимости, в которой они объявлены;
  3. const - объявляет локальную константу, т.е., константы существуют только в той области видимости, в которой они объявлены;
